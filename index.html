D-AMPS


(Digital-Advanced Mobile Phone System)
Employs the same 30 kHz channel spacing and frequency bands (824-849 and 869-894 MHz) as AMPS
Each voice channel is digitezed using complex PCM and compression techniques

GSM


(Global System for Mobile Communication
GSM standard originally described a digital, circuit-switched network optimized for Full Duplex Voice Telephony

Bands

  • Uses two bands for Duplex communication
  • Each being 25 MHz in width
  • Each further divided into 124 channels of 200 kHz

Transmission

IS-95

(Interim standard)
cdmaOne






IS-95 is the first CDMA based digital cellular technology
Code Division Multiple Access allows several radios to share the same Frequencies

Bands

  • 2 bands for Duplex communication
  • ISM 800-MHz bands or ISM 1900-MHz bands
  • Each band is divided into 20 channels of 1.228 MHz
  • 10 channels for each service provider
  • Each IS-95 channel is equivalent to 41 AMPS channels

Transmission

Forward Transmission

Base to Mobile

Reverse Transmission

Mobile to Base

synchronization

  • All base channels need to be synchronized to use CDMA
  • To provide synchronization, bases use the services of a satellite system (GPS)
Frequency reuse factor is 1 as the interferense from the neighboring cells cannot affect CDMA transmission
